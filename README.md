# ⚽ FIFA Player Data Analysis

## 📌 Overview

This project performs exploratory data analysis (EDA) on the FIFA player dataset to analyze player performance, salary distribution, and physical attributes. It focuses on understanding patterns across **shooting, defending, wages, nationality, and club-level data**.

---

## 📊 Dataset

* FIFA 20 Player Dataset (`players_20.csv`)
* ~18,000 players with 100+ attributes

Key features used:

* Shooting
* Defending
* Wage (EUR)
* Nationality
* Club
* Height & Weight (physical attributes)

---

## ⚙️ Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Google Colab

---

## 🔍 Key Analysis Performed

* Data loading from Google Drive
* Data cleaning and preprocessing
* Analysis of top nationalities by player count
* Salary analysis (top earning players)
* Sorting players based on wages and attributes
* Country-specific analysis (Argentina players)
* Club-level analysis (Liverpool players)
* Shooting and defending performance comparison
* Visualization using bar charts

---

## 📈 Key Insights

* Countries like England, Germany, and Spain have the highest number of players
* Top players earn significantly higher wages compared to average players
* Players like Lionel Messi and Cristiano Ronaldo dominate wage rankings
* Defenders such as Virgil van Dijk lead in defending stats
* Club-level analysis shows variation in player wages and performance
